## GET 全部分组列表

GET /v1/group/all-list

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "list": "string",
    "total": 0,
    "pageSize": 0,
    "currentPage": 0
  }
}
```
