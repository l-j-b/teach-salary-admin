## PUT 添加学生

PUT /v1/student/create

> Body 请求参数

```yaml
name: ""
gender: ""
avatar: ""
birthday: ""
age: 0
tags: ""
grades: ""
phone: ""
remarks: ""
wechart: ""
address: ""
email: ""
status: 0
groupId: ""

```

### 请求参数

| 名称         | 位置   | 类型     | 必选 | 说明         |
| ---------- | ---- | ------ | -- | ---------- |
| body       | body | object | 是  | none       |
| » name     | body | string | 是  | 名称         |
| » gender   | body | string | 否  | 性别         |
| » avatar   | body | string | 否  | 头像         |
| » birthday | body | string | 否  | 生日         |
| » age      | body | number | 否  | 年龄         |
| » tags     | body | string | 否  | 标签         |
| » grades   | body | string | 否  | 年级         |
| » phone    | body | string | 否  | 电话         |
| » remarks  | body | string | 否  | 备注         |
| » wechart  | body | string | 否  | 微信         |
| » address  | body | string | 否  | 住址         |
| » email    | body | string | 否  | 电子邮件       |
| » status   | body | number | 否  | 状态 1启用 2禁用 |
| » groupId  | body | string | 否  | 分组标识       |

> 返回示例

> 200 Response

```json

  "code": 0,
  "message": "string",
  "token": "string",
```
