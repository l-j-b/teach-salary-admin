## POST 添加课程

POST /v1/course/create

> Body 请求参数

```yaml
name: 测试课程名称
content: 课程内容
studentId: ""
remark: 备注
status: "1"
dataStatus: "1"

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|
|» name|body|string| 是 |课程名称|
|» content|body|string| 否 |课程内容|
|» studentId|body|[string]| 否 |学生标识集合|
|» remark|body|string| 否 |备注|
|» status|body|string| 否 |课程状态 1待开始(默认值) 3 进行中 3 已完成|
|» dataStatus|body|string| 否 |数据状态 1 启用 （默认值）0 禁用|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {}
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||接口状态值|
|» message|string|true|none||接口说明|
|» data|object|true|none||具体内容|
