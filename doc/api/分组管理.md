remark: ""

| 名称       | 位置   | 类型     | 必选 | 说明          |

| body     | body | object | 是  | none        |
| » name   | body | string | 否  | 名称          |
| » sort   | body | number | 否  | 排序          |
| » remark | body | string | 否  | 备注          |
